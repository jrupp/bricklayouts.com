Frequently Asked Questions
When can I start using BrickLayouts?
You can start using the early access version of BrickLayouts right now!
What platforms does BrickLayouts support?
Since BrickLayouts is a web application, you can access it from any device with a modern web browser.
How much does BrickLayouts cost?
Using BrickLayouts on your computer is and will remain free to use forever. In the future we will offer optional cloud-based features (like cloud storage, and live collaboration) through a paid subscription.
What features are planned for future releases? How can I suggest a feature?
We are constantly working on new features and improvements for BrickLayouts. You can view our development roadmap and suggest new features on our feedback board.
